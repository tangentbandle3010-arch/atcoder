## 3. 特定の条件の要素を消す: erase-remove
std::vector などから、特定の条件に合う要素を「効率的に」削除する手法。

- 使い方: v.erase(remove(v.begin(), v.end(), value), v.end());
- 数学的イメージ: 1. remove: 消したい要素を末尾に追いやり、有効な要素の「新しい境界線」を返す。2. erase: 境界線から末尾までの集合を物理的に削除する。

### 実装例：値が 3 の要素をすべて消す
```cpp
vector<int> v = {1, 3, 2, 3, 4};
// 3を末尾に寄せ、ゴミの開始地点（境界）から末尾まで削除
v.erase(remove(v.begin(), v.end(), 3), v.end());
// 結果: {1, 2, 4}
```
