# C++ & AtCoder 学習ノート

## 1. 文字列検索: find
文字列 $t$ の中から部分文字列 $s$ を探す関数。

- **使い方:** `t.find(s)`
- **戻り値:**
    - 見つかった場合：その開始位置のインデックス（$0, 1, 2, \dots$）
    - 見つからない場合：[npos](#2-不一致のシグナル-npos) を返す

### 実装例：隣り合う判定
```cpp
string t = "ACEGBDFAC";
string s; cin >> s;

// 見つかればYes、nposならNoを出力
cout << (t.find(s) != string::npos ? "Yes" : "No") << endl;
```

## 2. 不一致のシグナル: npos
`find` メソッドなどが「何も見つからなかった」時に返す特別な定数。

- **定義:** `static const size_t npos = -1;`
- **正体:** `size_t` 型（符号なし整数）の最大値。
- **注意点:** `int` 型の変数で戻り値を受け取ると、正しく比較できない場合がある。
- **参照:** [find関数の戻り値](#1-文字列検索-find)として利用される。

## 3. 特定の条件の要素を消す: erase-remove
`std::vector` などから、特定の条件に合う要素を「効率的に」削除する定番の手法。

- **使い方:** `v.erase(remove(v.begin(), v.end(), value), v.end());`
- **数学的イメージ:** 1. `remove`: 消したい要素を末尾に追いやり、生きてる要素だけの「新しい境界線」のイテレータを返す（写像の像を左に寄せる操作）。
  2. `erase`: 境界線から末尾までの「ゴミ」を物理的に削除する。

### 実装例：値が 3 の要素をすべて消す
```cpp
vector<int> v = {1, 3, 2, 3, 4};
// 3を末尾に寄せて、ゴミの開始地点から末尾まで削除
v.erase(remove(v.begin(), v.end(), 3), v.end());
// 結果: {1, 2, 4}
```

## 4. キーと値の対応付け: std::map による頻度集計
連想配列（map）を用いて、特定の要素が何回現れたかを記録・管理する手法。

- 概要: 文字や文字列などの「キー」に対して、出現回数という「値」を紐付けて保持し、動的に更新する。
- 数学的イメージ: 写像 $f: A \to \mathbb{Z}$ において、各要素 $x \in A$ に対応する値 $f(x)$ を逐次加算し、集合 $A$ 上の頻度分布を構築する操作。

### 実装例：文字列に含まれる各文字の頻度を数え上げる
```cpp
string s;
cin >> s;
map<char, int> mp;

// 文字を走査し、対応する整数の値をインクリメント
for (auto &nx : s) {
    mp[nx]++;
}
```

## 5. 順列の全列挙: next_permutation
与えられた要素の並び（順列）を辞書順で「次のパターン」に並べ替える関数。

- 概要: 現在の範囲を辞書順で一つ後ろの順列に変換する。全パターンを網羅するには、事前に範囲を [sort](#sort) して最小の順列にしておく必要がある。
- 数学的イメージ: 有限集合 $S$ の置換 $n!$ 通りを、辞書式順序という全順序に基づいて全走査する操作。

### 実装例：異なる順列をすべて生成し、その総数を求める
```cpp
string S = "aba";
sort(S.begin(), S.end()); // 辞書順最小から開始するためのソート
set<string> s;

do {
    s.insert(S); // 現在の順列を集合に記録
} while (next_permutation(S.begin(), S.end()));

// 結果: s.size() は重複を除いた順列の総数

```

## 6. 最低1回の実行を保証する: do-while文
条件判定をブロックの「末尾」で行うことで、中身の処理を必ず1回は実行させる制御構文。

- 使い方: do { /* 処理 */ } while (条件式);
- 数学的イメージ: 状態 $s$ に対する写像 $f$ の適用を先行させ、遷移後の状態 $f(s)$ が継続集合 $A$ に属するかを事後判定する反復。

### 実装例：[next_permutation](#5) との典型的な併用
```cpp
string s = "abc";
do {
    // 最初の状態（"abc"）を処理した後に、次の順列へ遷移させる
    cout << s << endl;
} while (next_permutation(s.begin(), s.end()));
// 判定が後ろにあるため、最初の並びも漏らさず処理できる
